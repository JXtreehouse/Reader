# Reader
> 一个简单的webApp书城整站
> 参考[webApp书城整站开发](http://coding.imooc.com/learn/list/13.html)

## 技术栈
- HTML5
- Vue.js
- Zepto	
- Node.js-Koa
- EJS

## Build setup
``` bash                                                 
# clone the repo into your disk.
$ git clone https://github.com/cccyb/Reader.git

# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ node app.js
```
